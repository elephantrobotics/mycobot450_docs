# 使用 myBlockly 前的准备工作

在下载 myBlockly 之前，您需要配置 Python 环境。详情请参考以下操作方法。

1. 环境配置。使用 myBlockly 之前，请确保已在计算机上配置好 Python 环境。

   > **Raspberry Jetson Nano 版本自带 Ubuntu（V-20.04）系统和内置 Python 开发环境，因此无需构建和管理。**
2. 软件配置。使用 myBlockly 之前，请确保已在计算机上安装好浏览器。
